## 预览
[预览地址](https://ruoyi-vue3-ts.netlify.app/)

## 介绍
根据RuoYi系统, 重写前端代码, 模版使用 vue-admin-template, 修改为 vue3+typescript, 完全使用RuoYi后端接口

## ⚙️ 环境
- node>=18
- pnpm>=7.30
- Vue Official>=2.1.10
- TypeScript>=5.0

## 技术栈
Vue.3 + TypeScript + Vite.5 + unocss + Axios + await-to-js

## 项目运行
1. 安装依赖
```
npm install
```
2. 运行
```
npm run serve
```
3. 打包
```
npm run build:prod
```
